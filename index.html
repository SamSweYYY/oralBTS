03. Projet 1

VTC Manager

React TypeScript Vite Tailwind CSS
  • Objectif : Application de gestion de chauffeurs VTC (CRUD complet).
  • Stack : React, TypeScript, Vite, Tailwind CSS.
  • Fonctionnalités : Inscription, connexion, profil, liste et suppression de chauffeurs.
  • Challenge : Authentification sécurisée et gestion des rôles.
  • Résultat : Interface moderne et responsive pour la gestion de flotte.
